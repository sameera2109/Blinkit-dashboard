###Blinkit-Dashboard
###  Title : BlinkIT Grocery Sales Analysis Dashboard
###  Description
This project analyzes BlinkIT grocery sales data using Power BI to uncover key business insights related to sales performance, product distribution, and outlet efficiency.The dashboard transforms raw transactional data into an interactive report that helps understand how different factors like outlet type, location, and product categories impact overall sales.
###  Tools & Technologies
- Power BI
- Power Query (Data Cleaning & Transformation)
- DAX (Data Analysis Expressions)
- Excel (Dataset)
###  Features & Highlights
- Interactive dashboard with dynamic slicers (Outlet Size, Location, Item Type)
- KPI cards displaying Total Sales, Average Sales, Number of Items, and Ratings
- Sales breakdown by Item Type, Fat Content, and Outlet Type
- Outlet establishment trend analysis over time
- Clean and structured layout for better business understanding
###  Key Insights
- Total sales reached $1.2M across 8,523 items
- Supermarket Type 1 generates the highest revenue among outlet types
- Tier 3 locations show strong sales performance compared to other tiers
- Balanced contribution between Low Fat and Regular products
- Certain product categories (like fruits, snacks, household items) drive higher sales
